# 高阶分析提示词模板

> 不是「帮我分析这份数据」，而是具体的、有难度的分析需求。
> 根据数据类型自动匹配，也可由用户直接指定。

## 使用方式

当用户的分析请求过于模糊（如「帮我分析这份数据」）时：
1. 先读取Excel概览，判断数据类型
2. 从下面匹配的分析模板中选取3-5个分析维度
3. **不问用户**，直接执行深度分析，让产出超预期

## 广告投放数据分析

### 推荐组合（展示最佳效果）
```
请对这份投放数据做深度分析，要求：
1. 渠道效率矩阵：用消耗-ROI散点图标注各渠道，找出被低估和需优化的渠道
2. ROI拐点分析：找出每个渠道的最优投放区间，标注消耗超过多少ROI开始下降
3. 时间模式挖掘：分析工作日vs周末、上午vs下午vs晚间的投放效率差异
4. 异常检测：用IQR方法识别异常数据点并给出业务归因
5. 预算优化建议：在总预算不变前提下，给出渠道重新分配方案和预期提升幅度
```

### 单项深度分析

**ROI拐点分析**
```
找出ROI拐点：日消耗在什么水平时ROI开始显著下降？
画出消耗-ROI散点图，拟合趋势线，标注拐点位置。
给出每个渠道的最优消耗区间建议。
```

**渠道归因与协同效应**
```
分析付费流量对自然流量的带动效应：
千川投放量增加时，自然流量是否同步增长？
计算每增加1万千川消耗，自然流量平均增加多少？
```

**异常检测与归因**
```
用统计方法识别所有异常数据点。
区分「好异常」（突破性增长）和「坏异常」（意外下跌），分别给出应对策略。
```

**预算优化建模**
```
在总预算不变的前提下，用边际ROI递减模型计算：
如何在各渠道间重新分配预算能最大化整体ROI？
给出「当前分配」vs「优化分配」的对比表。
```

**退货风险预警**
```
分析退货率趋势：哪些品类的退货率在持续恶化？
做敏感性分析：如果退货率再升5个百分点，哪些品类会从盈利变亏损？
```

## 销售数据分析

### 推荐组合
```
请对这份销售数据做深度分析，要求：
1. 增长归因：增长来自哪里——新客户、老客户复购、客单价提升、还是品类扩张？
2. 价格弹性：哪些商品提价后销量不变（刚性需求），哪些降价就暴增（价格敏感）？
3. 客户分层（RFM）：高价值/待激活/流失预警/新客各占多少？
4. 长尾诊断：有多少SKU贡献了80%的销售额？低效SKU是否在拖累整体利润率？
5. 季节性识别：哪些品类有明显的周期性波动？
```

## 用户行为数据分析

### 推荐组合
```
请对这份用户行为数据做深度分析，要求：
1. 转化漏斗：访问→注册→首次使用→付费→复购，找出最大流失环节
2. 留存曲线：日/周留存率，找出留存拐点（用户在第几天流失最严重）
3. 渠道质量：不同来源用户的LTV差异，找出性价比最高的获客渠道
4. 行为聚类：用户可分为几类？每类的典型行为模式是什么？
```

## 财务数据分析

### 推荐组合
```
请对这份财务数据做深度诊断，要求：
1. 盈利能力：毛利率、净利率趋势，与行业平均对比
2. 运营效率：资产周转率、应收账款周转天数，找出现金流风险
3. 杜邦分解：ROE变动的主要驱动因素是利润改善、效率提升还是杠杆变化？
4. 成本结构：固定成本vs可变成本比例，盈亏平衡点在哪里？
```

## 通用高阶分析模式

无论什么数据类型，以下模式都可叠加使用：

| 模式 | 一句话说明 | 分析难度 |
|------|-----------|---------|
| 对比分析 | A vs B，找差异找原因 | ⭐ |
| 趋势分析 | 随时间变化的走势和拐点 | ⭐ |
| 结构分析 | 各部分占比，二八法则验证 | ⭐ |
| 异常检测 | IQR/Z-score识别偏离值 | ⭐⭐ |
| 归因分析 | 增长/衰退的原因追溯 | ⭐⭐ |
| 敏感性分析 | 变量变化对结果的影响 | ⭐⭐⭐ |
| 预测分析 | 基于历史趋势的外推 | ⭐⭐⭐ |
