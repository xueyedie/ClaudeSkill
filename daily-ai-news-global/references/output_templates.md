# 输出格式模板 — 国际版（默认中文输出）

最终呈现给用户的内容必须为简体中文（链接保持原样；专有名词可保留英文/缩写）。

## 通用：每条新闻目录索引（必做）

**硬性要求**：
- **目录必须细到每条新闻**：在「📑 目录」下，按分类做二级嵌套，列出该组下每条新闻标题的链接
- **每条新闻必须有稳定锚点**：在每条新闻标题上方插入 `<a id="xxx"></a>`，目录用 `(#xxx)` 链接

**锚点命名**（同一份简报内唯一且连续）：
- 重要发布：`maj-01`、`maj-02`…
- 研究与论文：`res-01`、`res-02`…
- 产业与商业：`biz-01`、`biz-02`…
- 工具与应用：`tool-01`、`tool-02`…
- 政策与伦理：`pol-01`、`pol-02`…
- 技术大牛观点：`guru-01`、`guru-02`…

## 🔺 今日新闻五层架构分布

> AI 产业自下而上分为五层：能源 → 芯片 → 基建 → 大模型 → 应用。
> 越底层影响越大但变化越慢；越上层迭代越快但更易被替代。

```
                        ┌─────────┐
                       │ 🚀 应用  │ ← [N] 条
                      │  L5-应用  │
                     ├───────────┤
                    │ 🧠 大模型   │ ← [N] 条
                   │  L4-大模型   │
                  ├─────────────┤
                 │ 🏗️ 基建       │ ← [N] 条
                │  L3-基建       │
               ├───────────────┤
              │ 🔧 芯片          │ ← [N] 条
             │  L2-芯片          │
            ├─────────────────┤
           │ ⚡ 能源              │ ← [N] 条
          │  L1-能源              │
         └───────────────────┘
```
