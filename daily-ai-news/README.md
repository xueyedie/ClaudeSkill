## daily-ai-news 技能说明（中文）

该技能用于聚合多个来源的 AI 最新动态，并生成结构化的中文简报。

## 你会得到什么

- **最终输出**：一份中文 Markdown 简报（按分类组织，包含原文链接）
- **落盘文件**：生成后会把最终简报写入项目工作区的指定目录（见下文）

## 输出文件（必做）

- **输出目录**：`./daily-AI-News/Output/`
- **文件命名**：`daily-ai-news-YYYYMMDD.md`（YYYYMMDD 为生成日期，例如 `20260223`）
- **覆盖规则**：同一天重复生成会覆盖同名文件，保证文件内容为最新
- **文件内容**：必须与对话中最终展示的简报 Markdown 内容一致（不要额外包一层说明文字）

> 该输出目录内还有一份更简短的说明：`daily-AI-News/Output/README.md`（项目侧的输出约定以它为准）。

## 输出语言约束

- 面向用户的最终内容必须为**简体中文**（标题、摘要、要点、结论等）
- 专有名词（公司/产品/模型/论文名）可保留英文/缩写；必要时可在首次出现时补充中文解释
- 链接（URL）必须保持原样
